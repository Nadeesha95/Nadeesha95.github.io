Privacy Policy

Last Updated: 2025-03-19

This Privacy Policy explains how the YouTube Subtitle Translator Chrome Extension we handles your information. We are committed to protecting your privacy and ensuring transparency about our data practices.

1. Data Collection

  • Subtitles: The extension accesses YouTube subtitles to provide real-time translations. This data is processed locally and not stored or transmitted to any external servers.
  • Usage Data: The extension tracks daily usage counts to enforce usage limits. This data is stored locally using chrome.storage.sync and is not shared with third parties.
  • Preferences: User preferences (e.g., enabled/disabled state, selected language) are stored locally using chrome.storage.sync.

2. Data Usage

  • The extension uses subtitle data solely for the purpose of providing real-time translations.
  • Usage data is used to enforce daily usage limits and improve the user experience.
  • Preferences are used to customize the extension's behavior based on user settings.

3. Data Sharing

We do not share any user data with third parties. All data is processed locally and never transmitted to external servers.

4. Data Security

All data is stored securely using Chrome's storage.sync API, which encrypts data in transit and at rest. We do not collect or store personally identifiable information (PII).
